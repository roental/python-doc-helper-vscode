# Python Doc Helper

Автор: Курбанов Раджаб Мухтарович, группа M3119

## Описание плагина

Python Doc Helper — это расширение для Visual Studio Code, которое вставляет готовый шаблон функции с документированием на русском языке.

Плагин упрощает оформление лабораторных работ по документированию кода.

## Возможности

- Вставка шаблона Python функции с docstring одним действием.
- Структурированный шаблон: описание, параметры, пример вызова, результат.
- Сообщение об успешной вставке шаблона.

## Как использовать

1. Откройте любой `.py` файл в Visual Studio Code.
2. Установите курсор в то место, куда хотите вставить шаблон.
3. Нажмите `Ctrl + Shift + P`, чтобы открыть Command Palette.
4. Введите: `Insert Python Docstring Template`.
5. Нажмите Enter — шаблон функции будет вставлен в текущую позицию курсора.

## Структура проекта

- `src/extension.ts` — основной код плагина, регистрация команды и логика вставки шаблона.
- `package.json` — описание плагина, идентификатор, команда и метаданные.
- Прочие файлы — конфигурация сборки и тестов (созданы автоматически генератором VS Code).
